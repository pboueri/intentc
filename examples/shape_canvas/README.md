# Shape Canvas

A self-contained shape canvas application that renders geometric shapes on a 2D canvas. The canvas is divided into quadrants and displays three colored shapes — a red circle, a green triangle, and a blue square — positioned across the canvas to form a composed scene.

No external dependencies are required. The entire application is a single HTML file that runs directly in the browser.

## How to Run

```bash
./start.sh
```

This will open the application in your default browser and print the file path to the terminal.

## What You'll See

A 2D canvas divided into quadrants with three shapes:

- A **red circle**
- A **green triangle**
- A **blue square**

Each shape is positioned in a different area of the canvas.
