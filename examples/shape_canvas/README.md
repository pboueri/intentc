# Shape Canvas

A self-contained shape canvas application that renders geometric shapes on a 2D canvas divided into quadrants. The canvas displays three colored shapes — a red circle, a green triangle, and a blue square — positioned across the canvas to form a composed scene.

## How to Run

```bash
./start.sh
```

This starts a local HTTP server and prints the URL to the terminal. Open the URL in your browser to view the canvas.

## What You'll See

A 2D canvas divided into quadrants with three shapes:

- A **red circle**
- A **green triangle**
- A **blue square**

Each shape is positioned in a different area of the canvas, creating a simple composed scene.
